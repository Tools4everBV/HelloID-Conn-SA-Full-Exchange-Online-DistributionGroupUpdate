# Change Log

All notable changes to this project will be documented in this file. The format is based on [Keep a Changelog](https://keepachangelog.com), and this project adheres to [Semantic Versioning](https://semver.org).

## [1.1.0] - 20-05-2026

### Added

- Created configuration and input files for checking unique display names and email addresses for Exchange Online distribution groups.
- Implemented PowerShell scripts to validate the uniqueness of display names and email addresses against Microsoft Entra ID.
- Added functionality to retrieve all verified mail domains associated with the tenant.
- Mail-enabled Security Groups are now supported by default.
